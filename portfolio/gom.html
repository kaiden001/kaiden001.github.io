Cemetery Management System

A comprehensive solution designed to efficiently manage cemeteries by seamlessly handling various aspects of memorial spaces.

TECHNOLOGIES

FEATURES AND MAIN FUNCTIONS

DASHBOARD

The dashboard gives a complete view of all system data, making it easy to manage and analyze. It includes counts for total deceased, revenues, available blocks, overdue items, and reservations.

MAP

The system integrates maps to display blocks, each color-coded to represent its status, facilitating easy viewing of available options. This feature enhances user navigation and decision-making processes within the system.

MANAGE BLOCKS

Users can manage block data and information, allowing them to edit or update details related to the deceased and payor for specific blocks which enhanced user control and efficiency within the system.

GENERATE DOCUMENTS

The system offers an efficient document printing function for processes like applications, interment, payments, and FP certificates. Users can quickly generate and print documents, streamlining workflow and task processing.

PAYOR LIST

The system displays payor information, allowing users to view payors and the specific blocks they've paid for on a single page.

USER MANAGEMENT

The User Management page allows you to control user statuses effortlessly. With simple options to activate or deactivate users, it streamlines user management tasks for smooth operation.

LOCATIONAL CLEARANCE INFORMATION SYSTEM

A system designed to enhance efficiency, accessibility, and transparency in facilitating locational clearance transactions.

TECHNOLOGIES

FEATURES AND MAIN FUNCTIONS

DASHBOARD

The dashboard provides a comprehensive overview of all data in the system for easy and efficient management and analysis of data. It consist of charts,map for vizualization as well as view calendar for easy management of schedules.

ADVANCE TABLE FEATURES

The advance table features enables the user to filter data for easy searching. All the data about the applicants are shown through a table format, allowing for efficient data retrieval.

GENERATE DOCUMENTS

Enables users to generate documents for specific processes required for locational clearance. Various types of documents can be viewed and downloaded, and the system allows for the implementation and attachment of e-signatures to documents.

GENERATE QRCODE

Enables users to generate QR code for specific applicant, the qrcode can be downloaded and printed, also the qrcode will be automatically sent to the applicant's email.

QRCODE SCANNER

Scans QRCode of the applicant and automatically shows the applicant's current proccess and status, it allows the lcis administrator to assist the applicant more efficiently.

GEOTAGGING AND GEOLOCATION

Integration of Leaflet enables precise geotagging and geolocation functionalities, facilitating a smooth inspection process.

MANAGE USERS

Manage user's account details, including their roles, status, and other details. The system allows export of the user list in CSV format.

ACTIVITY LOGS

Tracks system data changes, such as additions of new applicants, approval of processes and etc. It displays the user, time, and date details of the user. Provides valuable insights for system monitoring and tracking

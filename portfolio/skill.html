Skill Registration System

A platform designed to enhance efficiency, accessibility, and transparency in connecting skills and program requests.

TECHNOLOGIES

FEATURES AND MAIN FUNCTIONS

DASHBOARD

The Dashboard presents key metrics such as registered and verified users, pending verifications, and total skills, with graphs for skill distribution, in-demand skills, and user demographics. It also highlights users with the most skills and recent registrants.

TALENTS

The Talent Page displays a list of registered users' talents, including their skills and certifications. You can show interest in a user's profile by clicking the heart button, which will instantly notify the user of your interest.

GLOBAL SEARCH

The Global Search feature allows users to find specific skills or individuals. For example, searching "cooking" will display users with cooking skills, and users can also search for specific people.

POSTS

The Post Page lets users create posts for specific skills, like painting or cooking. Interested users can click the heart button to notify the poster, who can then view interested users and check if their skills match the post's requirements.

PROGRAMS

The Programs Page allows the administrator to post programs, such as training sessions. Users can view these programs, express interest, and request to join. The administrator can view both interested users and those who have requested to join.

REQUEST JOIN LIST

The Request Join List page allows administrators to view users who have requested to join posted programs. Administrators can manage requests by accepting or rejecting them, with notifications sent to users regarding their application status.

REMINDERS

The Reminders page helps both administrators and users track program schedules. Administrators see upcoming programs, while users can view schedules for programs theyâ€™ve joined.

REPORTS

The system allows users to report posts they believe violate regulations. Administrators can review and verify these reports, update their status, or mark them as resolved on the Report Page.

REGISTERED SKILLS

The Registered Skills page allows administrators to manage all registered skills. They can view users associated with each skill and edit or remove skills from the system if needed.

REGISTERED USERS

The Registered Users page allows administrators to manage user accounts. They can change user statuses, adjust access levels, disable accounts, and view user profiles.

ACTIVITY LOGS

The Activity Logs page allows administrators to monitor user actions within the system. It shows the status of user activities, such as whether profile changes were successfully made.

ACCOUNT SETTING

The Account Settings page allows users to update their profile picture and change their account password.

NOTIFICATIONS

The system uses WebSocket to provide instant notifications. Users receive immediate updates when someone shows interest in their post or profile, or when a program join request is accepted or rejected.

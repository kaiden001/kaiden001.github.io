ORDERING SYSTEM

An integrated system designed to efficiently manage food orders by seamlessly handling menu browsing, order placement, and tracking, ensuring a smooth and user-friendly experience

TECHNOLOGIES

FEATURES AND MAIN FUNCTIONS

MENU

The Menu Page allows users to view all available products. Users can order or select their desired items and add them to their cart, as well as mark specific menus as favorites.

CART

The admin dashboard provides an overview and counts of total sales, Product status as well as number of products available. It also shows top selling products and recent transactions for easy viewing.

CHECKOUT

The Checkout Page is where users finalize their orders. Here, they confirm and place their order for the selected food or products.

ORDERS

The Orders Page allows users to monitor the status of their orders. Users can see if their order is being prepared or is on its way for delivery.

FAVORITES

The Favorites Page displays the food or products that users have marked as favorites. Users can easily add these items to their future orders.

ACCOUNT SETTING

The Account Settings page allows users to update their profile information, including their profile picture and account password.

ADMIN DASHBOARD

The admin dashboard offers an overview of total sales, product status, and the number of available products. It also highlights top-selling products and recent transactions for convenient viewing.

INVENTORY

The inventory page enables admins to manage product details and add new items. It includes a legend and sections summarizing product counts based on their status.

POINT OF SALES

The primary function of the POS page is to process transactions. It allows cashiers to input items for purchase, calculate the total amount due, apply discounts, and accept payments.

ADMIN ORDERS PAGE

The Admin Order Page allows administrators to manage customer orders. They can update the status of orders and view detailed information about each customer's order.

REPORTS

The Data Reports feature includes graphs based on sales data and allows users to filter information by days, months, and years.

GENERATE DOCUMENTS

The system can generate document reports on sales and inventory, which are suitable for printing.

EXPORT TO CSV

The system allows administrator to generate CSV files. This feature enables easy export of data for use in other applications or for further analysis.

USER MANAGEMENT

Manage user account details and add new users. This feature allows for updating existing account information and creating new user profiles within the system.

SYSTEM SETUP

Allows administrator to update the system’s logo, name, and other details. This feature enables customization of the system’s branding and settings.

INVENTORY SETTINGS

In the Inventory Settings page, administrators manage product details such as categories, prices, and discounts for specific items. This feature allows for comprehensive control over product information and pricing adjustments.

ACTIVITY LOGS

The Activity Logs feature provides a detailed record of user actions within the system. It tracks various activities, including changes and updates, to monitor and review user interactions.

MANAGE ARCHIVES

The system also allows administrators to manage archived data, including products and user accounts. This feature helps admins restore recent archives when needed.

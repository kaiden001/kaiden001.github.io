INVENTORY SYSTEM

System that enables the monitoring, management, and control of goods in stock, ensuring that operations maintain an appropriate inventory level

TECHNOLOGIES

FEATURES AND MAIN FUNCTIONS

DASHBOARD

The dashboard consist of overview of all data in the system for easy and efficient management and analysis of data. It also includes charts and counts of number of items base on their status.

INVENTORY

The inventory page helps users to monitor and manage items, it shows items in stock and their status. The system also allows users to filter/search items based on their status and category.

GENERATE QRCODE

The system also generates QRCodes for all the items, it can be used for searching and managing items.

SCANNER

The scanner page enables users to scan QRCode it has two types first is the scanning of QRCode using the device camera, second is through scanner machine.

REQUEST ITEM

The request item page enables users to request items from the system.

MANAGE REQUEST

Admin may manage pending request and approve/reject request. The request list are shown using data table which allows the admins to sort and search for a specific request.

GENERATE DOCUMENTS

Allows user to generate documents for a specific type of request.

USER MANAGEMENT

Manage user account details, including their status as well as add a new user.

SYSTEM SETUP

Allows user to change the logo, name and primary color of the system.
